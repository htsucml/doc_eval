\section{Introduction}

Document vision-language models (VLMs) are often evaluated with aggregate
question-answering accuracy, but deployment failures frequently involve
unsupported answers: the model supplies a plausible value even when the requested
field is absent or unreadable. This repository packages a bounded take-home
evaluation for small document VLMs with an explicit focus on reproducibility and
abstention behavior.

The primary target models are sub-1B systems, especially SmolVLM 500M and
SmolVLM2 500M. Larger systems, including SmolVLM2 2.2B and Qwen2.5-VL 3B, are
reported only as reference models. This distinction matters: the report uses
larger references to contextualize scale and ceiling behavior, not to replace
the compact-model target comparison.

The final packaged artifacts include tests, CPU-safe smoke evaluation, GPU mini
reproduction commands, aggregation scripts, and a paper build target. The paper
tables are generated from existing CSV files under \texttt{reports/}.
