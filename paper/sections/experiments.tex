\section{Experiments}

The primary sub-1B target models are SmolVLM 500M and SmolVLM2 500M Video. The
main reference-only models are SmolVLM2 2.2B, a same-family $>$1B reference, and
Qwen2.5-VL 3B, an external $>$1B ceiling. All real-model runs are gated by
\texttt{--allow-real-models}; CPU tests and smoke runs use the deterministic
dummy adapter.

The LoRA/SFT component is a bounded methodology proof of concept for SmolVLM2
500M only. It uses separate synthetic training data and does not train on
DocMiniBench-v0, controlled NOT\_FOUND, manual real-document NOT\_FOUND, or OOD
sanity evaluation rows.
